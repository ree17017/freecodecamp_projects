Introduction

Creating your first character shiould be fun and exciting. Don't stress if it perfect or not this will not be the only character you build.

Pick a name and maybe write a background for your caracter. Keep your background to a few pages unless you have RPGed this out with your DM.

Race Selection

Read though each race description and pick something that sounds like you. It is better to start with something you like or know and then build into more difficult races as you gain more skill.

It can be a lot of fun to pick a non-standard race. Word of advice. If this is your first character pick a main line race.

Class Selection
  • >Place 4

    • Class selection can be hard and intemidating chore. I would stick to a melee class or cleric as my first time class. Spell casters like the Wizard are fun classes but take a little bit of a learning curve. If you like to learn more about your class than others have to then you could play a wizard or sourcer.

    Fighter, Barb, Rogue, and Ranger are some of the easiest classes to start playing. Bash down the door and attack the closest bad guy can be fun!
